# Change: เพิ่มระบบโพสต์หางานทั่วไป

## Why
เพื่อส่งเสริมเศรษฐกิจในพื้นที่หาดใหญ่หลังน้ำท่วม โดยเปิดโอกาสให้คนที่ตกงานหางานพิเศษทำได้ และคนที่ต้องการจ้างงานทั่วไปสามารถโพสต์หาคนทำได้ง่ายขึ้น

## What Changes
- เพิ่มฟีเจอร์โพสต์หางานทั่วไปแบบ 2 ฝั่ง (คนหาคนทำ และ คนหางานทำ)
- รองรับงานทั่วไปนอกเหนือจากงานซ่อม/ล้าง/ขนย้ายที่มีอยู่แล้ว
- ให้ผู้โพสต์สามารถระบุเรทค่าจ้างได้
- ระบบการติดต่อระหว่างคนโพสต์และคนสนใจโดยตรง
- **BREAKING**: เปลี่ยนโครงสร้างข้อมูลงานจาก 3 ประเภท (ซ่อม/ล้าง/ขนย้าย) เป็นระบบแบบ flexible categories

## Impact
- Affected specs: marketplace, user-profile
- Affected code: app/dashboard/jobs, components/jobs, lib/database/jobs
- Database schema: ต้องเพิ่มประเภทงานทั่วไปและฟิลด์เพิ่มเติม
- UI: เพิ่มหน้าโพสต์งานใหม่ และฟอร์มกรอกข้อมูลแยกระหว่าง 2 ฝั่ง
